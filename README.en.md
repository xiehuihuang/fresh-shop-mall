# freshShopMall

#### Description
生鲜电商：1.项目采用HbuilderX + uni-app + vue.js + flex弹性响应布局开发一套代码打包成移动app（ios、安卓app）、微信小程序、h5网站; 2. 业务功能模块：注册（手机+短信、第三方微信授权）、登录、生鲜电商首页、商品分类、商品秒杀抢购活动、商品详情、商品用户评价、购物车、下订单支付（目前只支持微信和支付宝支付）、用户模块、订单管理模块、收藏商品;
